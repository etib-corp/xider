# Android Platform Implementation

This directory contains the implementation for running XIDER on Android mobile platforms. The code in this directory is designed to work with Android's unique input and display configurations, ensuring that users can enjoy XIDER on their mobile devices without any issues.

## Building for Android

We use Android Studio and the Android NDK to build the XIDER project for Android. The CMakeLists.txt file in this directory is configured to include the necessary libraries and dependencies for Android development.
