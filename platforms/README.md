# XIDER Platforms

This directory contains platform-specific implementations for the two main platforms we are targeting: **Windows**, **macOS**, and **Linux** and **Android**. Each platform has its own subdirectory with the necessary code and configurations to run the XIDER Project on that platform.

## List of Platforms

- [Linux](./linux/README.md), this directory contains the implementation for running XIDER on Linux desktop environments.

- [macOS](./macos/README.md), this directory contains the implementation for running XIDER on macOS desktop environments.

- [Windows](./windows/README.md), this directory contains the implementation for running XIDER on Windows desktop environments.

- [Android](./android/README.md), this directory contains the implementation for running XIDER on Android mobile platforms.
